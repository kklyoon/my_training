## 타일 장식물 (lv.3)

[문제 링크](https://programmers.co.kr/learn/courses/30/lessons/43104)


그냥 피보나치 수열 문제 이게 왜 레벨3 문제인지 모르겠다.

의미가 있다면 피보나치 수열을 그림그려서 보여준다 정도?

```
long long solution(int N) {
    long long answer = 0;
    long long a = 1, b = 1, c;
    for(int i = 3 ; i <=N ; i++){                
        c = b;
        b = a+b;
        a = c;
    }
    answer = (a+b+b)<<1;
    return answer;
}
```
